w2ui

w2ui 是一个jQuery UI 库，包括了许多控件用于创建类似于桌面应用和数据驱动的Web程序。 它包含了以下控件：layout, sidebar, toolbar, grid, popup, tabs, forms 和 field controls。这些控件采用HTML5-CSS3技术实现，并拥有强大的API来控制它们的各个方面。
