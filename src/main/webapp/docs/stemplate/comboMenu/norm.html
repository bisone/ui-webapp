统一标准

1.Bootstrap

CSS 和 JS 框架统一标准参照Bootstrap;
设置全局 CSS 样式基本的 HTML 元素均可以通过 class 设置样式并得到增强效果；还有先进的栅格系统。
组件无数可复用的组件，包括字体图标、下拉菜单、导航、警告框、弹出框等更多功能。
JavaScript 插件jQuery 插件为 Bootstrap 的组件赋予了“生命”。可以简单地一次性引入所有插件，或者逐个引入到你的页面中。

官网：http://v3.bootcss.com/css/
